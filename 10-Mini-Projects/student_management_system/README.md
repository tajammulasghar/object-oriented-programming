# Student Management System

A beginner-friendly OOP project for managing student records and marks.

## Features

- Create student records with roll number and name
- Store and manage three subject marks
- Calculate average marks
- Display student information

## Files

- `student_system.cpp` - Complete implementation with main function

## How to Compile and Run

```bash
g++ -o student_system student_system.cpp
./student_system
```

## Example Output

```
=== Student Management System ===

--- Student Records ---
Roll: 1, Name: Alice, Marks: 85 90 88, Average: 87.6667
Roll: 2, Name: Bob, Marks: 75 80 78, Average: 77.6667
Roll: 3, Name: Charlie, Marks: 92 95 90, Average: 92.3333
```
