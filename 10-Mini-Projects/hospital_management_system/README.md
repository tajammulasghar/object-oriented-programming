# Hospital Management System

A beginner-friendly OOP project for managing patient records and hospital operations.

## Features

- Create patient records with ID, name, disease, doctor, and room number
- Store and retrieve patient information
- Assign doctors to patients
- Update doctor assignments
- Display patient details

## Files

- `hospital_system.cpp` - Complete implementation with main function

## How to Compile and Run

```bash
g++ -o hospital_system hospital_system.cpp
./hospital_system
```

## Example Output

```
=== Hospital Management System ===

--- Patient Records ---
ID: 1001, Name: Alice, Disease: Fever, Doctor: Dr. Smith, Room: 101
ID: 1002, Name: Bob, Disease: Headache, Doctor: Dr. Jones, Room: 102

--- Update Doctor ---
Doctor updated to Dr. Wilson
```

## Concepts Used

- Classes and Objects
- Encapsulation (private members)
- Constructors with multiple parameters
- Getter and Setter methods
- String manipulation
- Object management
