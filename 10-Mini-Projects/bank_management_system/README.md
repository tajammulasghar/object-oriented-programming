# Bank Management System

A beginner-friendly OOP project demonstrating bank account operations.

## Features

- Create accounts with ID, name, and balance
- Deposit money with validation
- Withdraw money with balance checking
- Display account details
- Error handling for invalid operations

## Files

- `bank_system.cpp` - Complete implementation with main function

## How to Compile and Run

```bash
g++ -o bank_system bank_system.cpp
./bank_system
```

## Example Output

```
=== Bank Management System ===
ID: 101, Name: Alice, Balance: 5000
ID: 102, Name: Bob, Balance: 3000

--- Account 1 Operations ---
Deposit successful
ID: 101, Name: Alice, Balance: 6000

Withdrawal successful
ID: 101, Name: Alice, Balance: 4000
```
